ACM-ICPC 2016 大连

A - Wrestling Match

  • 题意:给出n个人,m个敌对关系,x个好人,y个坏人,问能否 确定地 将所有人划为好人/坏人。
  • 其中 $n\leq 1000 , m\leq 100000$
  • 常规做法:直接DFS/BFS,对同一阵营的染色.
  • 我的做法: 这题和种类并查集太像了,后来思考了一下的确可以用并查集做。具体做法类似NOIp原题关押罪犯,唯一的难点在于最后检测，我原本是检测剩下的独立集个数<=2，但显然不符合5 0 3 0, 2 0 0 0这种数据。实际上我在做的时候用了enemy数组记录每个集合的敌人集合,所以只需在最后检测每个集合是否有自己的敌对集合,如果没有则是无法判定的集合。当然,矛盾的处理在之前就已经完成。
